# Week 1 - Mobile Robotics Lab

The lab session provided an introduction to basic ROS 2 concepts through three main activities which included creating workspaces and developing software packages and running software components.

## Commands Used
- ros2 pkg create
- colcon build
- ros2 run
- ros2 pkg list

## Problems Faced
The workspace install folder did not exist at the beginning. The issue got resolved through a workspace rebuild which used colcon build command.

## Reflection
The lab session demonstrated to me how ROS 2 software organizes its components into separate nodes and complete software packages. I learned the importance of sourcing the workspace and registering entry points correctly. The debugging process helped me develop confidence in using Linux terminal commands together with project structure management.
